Here's how it works

Train the model

Add your transcripts and question answer files to train the model. The model creates a framework from your added transcript files which is converted to most likely question answers pair to help model more likely to answer as you do.

Generate the response and give feedback

After you have trained the model you can generate the response. If you generate the response without training the model it is likey that you will get the random answer. You can give feedback to the model if the response is not correct or likely to be correct.
